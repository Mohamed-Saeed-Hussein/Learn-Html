## SLIDE 26: FORMS - INTRODUCTION
**Forms collect user input:**

```html
<form action="/submit" method="post">
    <!-- Form elements go here -->
</form>
```

**Form attributes:**
- `action` - Where to send the data
- `method` - How to send data (GET or POST)
- `name` - Form identifier

**Why forms matter:**
- User registration
- Contact forms
- Search functionality
- Data collection

---

## SLIDE 27: INPUT TYPES - TEXT
**Text input for single-line data:**

```html
<input type="text" name="username" placeholder="Enter username">
<input type="email" name="email" placeholder="Enter email">
<input type="password" name="password" placeholder="Enter password">
<input type="url" name="website" placeholder="Enter website URL">
```

**Common attributes:**
- `name` - Identifies the field
- `placeholder` - Hint text
- `value` - Default value
- `required` - Must be filled

---

## SLIDE 28: INPUT TYPES - NUMBERS AND DATES
**Specialized input types:**

```html
<input type="number" name="age" min="0" max="120">
<input type="date" name="birthday">
<input type="time" name="meeting-time">
<input type="tel" name="phone" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}">
```

**Number attributes:**
- `min` - Minimum value
- `max` - Maximum value
- `step` - Increment value

---

## SLIDE 29: CHECKBOXES AND RADIO BUTTONS
**Multiple choice and single choice:**

```html
<!-- Checkboxes (multiple choice) -->
<input type="checkbox" name="interests" value="sports"> Sports
<input type="checkbox" name="interests" value="music"> Music
<input type="checkbox" name="interests" value="reading"> Reading

<!-- Radio buttons (single choice) -->
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<input type="radio" name="gender" value="other"> Other
```

**Important:** Radio buttons with same `name` work as a group

## SLIDE 30: SELECT DROPDOWNS
**Dropdown selection menus:**

```html
<select name="country">
    <option value="">Choose a country</option>
    <option value="us">United States</option>
    <option value="ca">Canada</option>
    <option value="uk">United Kingdom</option>
</select>
```

**Multiple selection:**
```html
<select name="colors" multiple>
    <option value="red">Red</option>
    <option value="blue">Blue</option>
    <option value="green">Green</option>
</select>
```

## SLIDE 31: TEXTAREA
**Multi-line text input:**

```html
<textarea name="message" rows="4" cols="50" placeholder="Enter your message here">
</textarea>
```

**Attributes:**
- `rows` - Number of visible rows
- `cols` - Number of visible columns
- `placeholder` - Hint text
- `maxlength` - Maximum characters

---

## SLIDE 32: LABELS
**Connect labels to form controls:**

```html
<!-- Method 1: Wrapping -->
<label>
    Username: <input type="text" name="username">
</label>

<!-- Method 2: Using 'for' attribute -->
<label for="email">Email:</label>
<input type="email" id="email" name="email">
```

**Benefits:**
- Screen readers understand the connection
- Clicking label focuses the input
- Better accessibility

---

## SLIDE 33: FORM VALIDATION
**Built-in validation attributes:**

```html
<input type="text" name="username" required>
<input type="email" name="email" required>
<input type="number" name="age" min="18" max="65">
<input type="text" name="zipcode" pattern="[0-9]{5}">
```

**Validation attributes:**
- `required` - Field must be filled
- `min` / `max` - Number range
- `pattern` - Regular expression
- `maxlength` - Character limit

## SLIDE 34: FIELDSET AND LEGEND
**Group related form fields:**

```html
<fieldset>
    <legend>Personal Information</legend>
    <label>Name: <input type="text" name="name"></label><br>
    <label>Email: <input type="email" name="email"></label><br>
    <label>Phone: <input type="tel" name="phone"></label>
</fieldset>
```

**Uses:**
- Organize long forms
- Create logical sections
- Improve accessibility

## SLIDE 37: HEADER AND FOOTER
**Page header and footer sections:**

```html
<header>
    <h1>Website Title</h1>
    <nav>
        <a href="home.html">Home</a>
        <a href="about.html">About</a>
        <a href="contact.html">Contact</a>
    </nav>
</header>

<footer>
    <p>&copy; 2024 My Website. All rights reserved.</p>
</footer>
```

**Uses:**
- `<header>` - Introduction content
- `<footer>` - Closing content

## SLIDE 38: NAVIGATION
**Navigation menus:**

```html
<nav>
    <ul>
        <li><a href="home.html">Home</a></li>
        <li><a href="about.html">About</a></li>
        <li><a href="services.html">Services</a></li>
        <li><a href="contact.html">Contact</a></li>
```

## SLIDE 39: MAIN CONTENT
**Primary content area:**

```html
<main>
    <h1>Main Article Title</h1>
    <p>This is the main content of the page...</p>
</main>
```

**Important rules:**
- Only one `<main>` per page
- Contains the primary content
- Excludes navigation, headers, footers
